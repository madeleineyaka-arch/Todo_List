# To-Do List Application

Cette application permet de gérer une liste de tâches.  
L’utilisateur peut ajouter, modifier, supprimer et marquer des tâches comme terminées.
Le projet est développé avec Spring Boot pour le backend,
MySQL pour la base de données et HTML / Tailwind CSS / JavaScript.

 1-Technologies utilisées

- Java
- Spring Boot
- Spring Data JPA
- MySQL
- HTML
- Tailwind CSS
- JavaScript

2- Fonctionnalités

- Ajouter une tâche
- Afficher toutes les tâches
- Modifier une tâche
- Supprimer une tâche
- Marquer une tâche comme terminée

3- Lancer le projet

1. Cloner le projet depuis GitHub
2. Créer la base de données `todolist_db` dans MySQL
3. Lancer l’application avec `TodolistApplication.java`
4. Ouvrir dans le navigateur :

http://localhost:8080
